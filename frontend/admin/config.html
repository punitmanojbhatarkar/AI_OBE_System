🚀 Phase 2: Cloud Migration Roadmap

After the 2-month pilot, the system will be migrated to a production cloud setup:

1
PostgreSQL Database — Replace localStorage with a full relational DB for multi-user concurrent access.
2
AWS S3 Storage — Store student Excel files, generated PDFs, and course audit logs in cloud storage.
3
Live AI API — Connect Gemini/Claude API for assignment generation and smart CO suggestions.
4
Multi-Tenant Rollout — Allow multiple colleges/institutions to use the platform concurrently.
5
Automated NBA Reports — Scheduled semester-end export of all NAAC/NBA Excel files with one click.
